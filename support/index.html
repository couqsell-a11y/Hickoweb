Computerhulp nodig

Computer op afstand overnemen?

Als bestaande klant kan je meegenieten van onze Snelle Support.
Hierbij kan een medewerker van Kockel IT jouw pc vanop afstand overnemen om je vraag of probleem vanop afstand op te lossen.
De Snelle support prestaties worden gefactureerd als gewone werkuren.
Start hier of bel eerst een medewerker van Kockel IT voor begeleiding.

Hoe Opstarten?

Om de Snelle Support te starten ga je als volgt te werk:
  • Maak een keuze uit onderstaande knoppen volgens jouw toestel.
  • Je mag de geselecteerde knop aanklikken. Er zal een download van het hulpprogramma opstarten (Splashtop SOS)
  • Dit bestand mag je uitvoeren waarna er een code op het scherm verschijnt.
  • Geef deze unieke code telefonisch door aan de Kockel IT medewerker waardoor wij jouw computer kunnen overnemen.
  • Je krijgt een melding dat iemand van Kockel IT je pc overgenomen heeft. Jij kan de sessie meevolgen indien gewenst.
